# 2026 Satellite project code 

this is the code from my 2026 FEPS satellite project. its a Sensor Aperature Radar (SAR) [1] implementation and uses the following components
* HC-SR04
* BMP280
* LDR
* LED
* Arduino Nano


## Bibliography 
[1] 	K. Baynes, "Synthetic Aperature Radar," National Aeronautics and Space Administration, 25 March 2026. [Online]. Available: https://www.earthdata.nasa.gov/learn/earth-observation-data-basics/sar. [Accessed 27 March 2026].
